Release log.

Every major iteration of TASS, newest first. TASS is in active development on the road to 1.0, and we version what we ship as we go.

Versioned, not dated. The number is the reference.

0.2.6Report bundleLatest

Finish an analysis and walk away with a complete, citable report in one step.

  • One-step report export that bundles a formatted HTML report, a reusable TASS project file, a Word document, and a PDF.
  • Report export wired directly into the desktop Export dialog.
  • Brought forward onto the native desktop shell with the full test suite passing.
0.2.0Native desktop shell

TASS moved to a proper native desktop application.

  • Rebuilt around a native Windows desktop shell for a faster, more stable workspace.
  • Added local-first error reporting that prepares a report for you to send and never transmits anything on its own.
  • Refined the analysis workflow with clearer in-app feedback and steadier panel behavior.
0.1.5Methods complete

The statistical core that makes TASS publication-ready.

  • Structured group comparisons with assumption checks, post-hoc tests, effect sizes, and confidence intervals.
  • Refined licensing and bundled-dictionary handling for clean, auditable provenance.
  • Hardened the analysis engine and fixed core crashes surfaced in heavier workloads.
0.1.0First build

Where it began: no-code dictionary scoring on your own machine.

  • The full interface, core analysis services, and the first round of bundled open dictionaries.
  • Dictionary-based text scoring with the supporting test suites in place from the start.
  • Established TASS branding and the local-first, offline foundation.
