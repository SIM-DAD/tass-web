How do I activate my license?

When TASS launches for the first time it will show an activation prompt. Paste the license key exactly as it appears in your receipt email — including any hyphens — and click Activate. TASS requires an internet connection for this one-time check; after activation it runs fully offline. If the key is rejected, confirm you are using the email address you purchased with and that there are no extra spaces in the pasted key. If you can’t find your key, email support with your order ID.

Can I move TASS to a new computer?

Yes. Individual licenses allow activation on up to two machines simultaneously. If you are replacing a machine, email support@simdadllc.com with your order ID and we will reset one of your activations at no charge, usually within one business day. To transfer your projects, copy the TASS Projects folder from your Documents directory on the old machine to the same location on the new one.

What file formats does TASS accept?

TASS accepts CSV, XLSX, and plain TXT files. CSV files must be UTF-8 encoded — if your file was exported from Excel, re-save it using File → Save As → CSV UTF-8 (Comma delimited) before importing. The column-mapping wizard lets you designate your text, group, and metadata columns after the file loads. Trial limitation: The free trial processes a maximum of 500 rows per analysis run. Upgrade to a full license to remove this limit. See Known Limitations below.

How do I cite TASS in a paper?

Every TASS export includes a pre-formatted APA citation block in a separate sheet or footer. You can also copy it at any time from inside the app: go to Help → Cite TASS. The cite dialog includes a separate citation block for each bundled dictionary used in the analysis — copy whichever ones apply to your methods section. If you need a BibTeX or other format, email support.

My antivirus flagged the installer — is it safe?

Yes, TASS is safe. Antivirus programs and Windows SmartScreen sometimes flag newly released executables that have not yet accumulated a large install history — this is a false positive, not an indicator of malware. TASS is digitally signed by SIM DAD LLC. To verify independently: check that the file’s SHA-256 hash matches the one published on the download page, and confirm the digital signature on the installer by right-clicking the file → Properties → Digital Signatures. If your institution’s IT policy blocks the installer entirely, email support@simdadllc.com and we can provide documentation for your IT department.

Known Limitations

What to Know
Before You Run

Trial

500-Row Analysis Limit

The 14-day free trial processes a maximum of 500 rows per analysis run. Rows beyond that are excluded from results. There is no limit on the number of trial analyses or the number of dictionaries you can test. A full license removes the row cap entirely. To purchase, visit the buy page.

Platform

Windows Only at Launch

TASS v1.0 is a native Windows application requiring Windows 10 version 2004 (build 19041) or later, 64-bit. Linux support is planned for v1.1 (Fall 2026). macOS support is under evaluation. Email support@simdadllc.com to be notified when additional platforms are available.
